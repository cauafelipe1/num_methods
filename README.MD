# num_methods

C++ and octave based repository for academic studies related to Numerical Methods and it's algorithms

## C++

### Features
- **Linear Systems**
  - Gaussian Elimination
  - Cholesky decomposition

### Requirements
- C++17 compiler (GCC/Clang)
- CMake 3.10+

### Build

```bash
# on cpp dir
mkdir build && cd build
cmake ..
make
```
